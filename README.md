# AWS Portfolio Website

## Purpose
This repository contains my portfolio website, built to present my AWS
and architecture projects in a clear, recruiter-friendly format.

## What the site includes
- Hero section with target role and direction
- About section linking IS degree to cloud architecture path
- Project showcase with six cards (portfolio + five AWS projects)
- Certifications section with badge-style cards
- Architecture diagrams for each project
- Contact links

## Stack
HTML, CSS, JavaScript

## Hosting
Amazon S3 (static website hosting) + Amazon CloudFront (HTTPS + CDN)

## Deployment
Initially deployed manually. Automated via GitHub Actions (deploy.yml)
after Project 4 is complete — every push to main triggers a deployment.

## AWS services used
- Amazon S3 (free tier)
- Amazon CloudFront (always free tier)

## Cost
Within AWS Free Tier. Total cost: $0
